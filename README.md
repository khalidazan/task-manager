# Simple Task List

A command-line task manager written in Python. Manage your homework or study tasks with persistent storage in a JSON file.

## Features 
- Add new tasks
- View all tasks with status
-  Mark tasks as done
-  Save tasks manually to tasks.json
-  Load tasks from file -
-  Automatic save on exit
## Requirements
-  Python 3.6 or higher
-   No external libraries (uses only standard library)
## Usage 
  1. Clone the repository or download tasks.py.
  2. Run the script:
```bash
python tasks.py
